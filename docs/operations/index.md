# Operations

Daily management, monitoring, and troubleshooting of Pi Gateway.

## Daily Tasks

- **[Daily Operations](daily-operations/)** — VPN client management, monitoring, backups
- **[Monitoring & Observability](monitoring/)** — Key metrics, logs, dashboards

## Common Operations

**Manage VPN Clients**
- Add new client
- Revoke client access
- Export client configuration
- Monitor active connections

**Monitor System Health**
- Check service status
- View logs
- Monitor resource usage
- Alert on failures

**Handle Issues**
- SSH connection problems
- VPN connectivity issues
- Firewall lockout recovery
- System performance

**Keep System Secure**
- Apply security updates
- Manage SSH keys
- Review access logs
- Audit firewall rules

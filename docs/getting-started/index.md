# Getting Started

Installation, setup, and first steps with Pi Gateway.

Choose your installation method:

- **[Quick Start](quick-start/)** — One-command installation (15 minutes)
- **[Manual Setup](setup/)** — Detailed step-by-step guide
- **[Local Development](local-development/)** — For developers and testers

## System Requirements

**Hardware:**
- Raspberry Pi 4 or newer (Pi 5 recommended)
- 32GB+ MicroSD card
- Reliable power supply
- Network connection (Ethernet or WiFi)

**Software:**
- Raspberry Pi OS (Lite or Desktop)
- SSH client installed locally
- Administrator access to your router

## What You'll Have After Setup

After completing setup, you'll have:

✅ **SSH Access** — Secure remote shell to your Pi
✅ **VPN Tunnel** — Encrypted access to home network
✅ **Firewall** — Hardened network security
✅ **Dynamic DNS** — Domain name that stays in sync
